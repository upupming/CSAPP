```c
for(int i = 0; i < 16; i++){
    for(int j = 0; j < 16; j++){
        square[i][j].y = 1;
    }
}
```

    square[0][0].y = 1

    square[0][1].y = 1

(2) 第二个循环中  

```c
for(int i = 0; i < 16; i++){
    for(int j = 0; j < 16; j++){
        square[i][j].c = 0;
        square[i][j].m = 0;
        square[i][j].k = 0;
    }
}
```

    square[0][0].c = 0

不命中，缓存两个结构体

    square[0][0].m = 0
    square[0][0].k = 0
    square[0][1].c = 0
    square[0][1].m = 0
    square[0][1].k = 0

命中  

后续如此循环，不命中的写总数为: 16 * 16 * 3 / 6 = 128
